Perguntas Frequentes

Respostas às dúvidas mais comuns sobre a nossa hospedagem WordPress.

O LiteSpeed é um servidor web de alto desempenho que substitui o Apache e é até 6× mais rápido para sites WordPress. Inclui cache integrada (LSCache), suporte a HTTP/3, compressão Brotli e otimizações específicas para WordPress que reduzem drasticamente os tempos de carregamento e melhoram a classificação no Google.
O WordPress Toolkit Deluxe é um painel avançado para gerir todas as suas instalações WordPress num único lugar. Permite instalar, clonar, sincronizar e fazer backup de sites com um clique. Inclui também funcionalidades de staging, gestão de atualizações automáticas e verificação de segurança integrada.
O Imunify360 é uma suite de segurança completa que protege o seu WordPress contra malware, ataques de força bruta, injeções SQL e vulnerabilidades conhecidas. Inclui firewall de aplicações web (WAF), scanner de malware em tempo real, proteção contra bots e sistema de reputação de IPs. É considerado um dos melhores sistemas de segurança para hospedagem web.
