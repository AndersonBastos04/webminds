# 🌐 Meu Site de Serviços de Hospedagem

Este projeto é um site completo para um serviço de hospedagem e tecnologia, desenvolvido com **HTML, CSS e JavaScript**, pronto para ser hospedado usando **GitHub Pages**. O site inclui diversas páginas informativas sobre serviços, parceiros, backups, SEO e muito mais.

---

## 📂 Estrutura do Projeto


/assets/ → Imagens, ícones e outros recursos estáticos
index.html → Página inicial
alojamento-cpanel.html → Página de hospedagem cPanel
alojamento-wordpress.html → Página de hospedagem WordPress
backups.html → Página sobre backups
cloud.html → Página de serviços de Cloud
criacao-website.html → Página de criação de sites
dedicados.html → Página de servidores dedicados
dominios.html → Página sobre domínios
monitorizacao.html → Página sobre monitorização
nordvpn.html → Página sobre NordVPN
parceiros.html → Página de parceiros
seo.html → Página sobre SEO
servidores-jogos.html → Página sobre servidores para jogos
sitelock.html → Página sobre SiteLock
sobre-nos.html → Página sobre a empresa
torne-se-afiliado.html → Página de afiliados
trabalhe-connosco.html → Página de carreiras
vps-nvme.html → Página sobre VPS NVMe
weebly.html → Página sobre Weebly
navbar.js → Script de navegação
script.js → Script principal
styles.css → Estilos gerais do site
README.md → Este arquivo

## 🚀 Como Visualizar o Projeto

O site pode ser acessado diretamente usando **GitHub Pages**:

1. Vá para: `https://andersonbastos04.github.io/nome-do-repositorio/`
2. Certifique-se de que o GitHub Pages está habilitado na branch `main` (raiz `/`).
3. Abra `index.html` para iniciar o site.

> 💡 Se você fizer alterações, basta dar commit e push no GitHub. O site será atualizado automaticamente após alguns minutos.
