Perguntas Frequentes

Um domínio é o nome que identifica de forma única uma pessoa, empresa ou organização na Internet. Por exemplo, o nosso domínio é webminds.pt. O domínio é composto por duas partes separadas por um ponto: o nome propriamente dito e uma extensão que pode indicar o tipo de atividade ou área geográfica (ex: .pt para Portugal, .de para Alemanha, .com para atividades comerciais).
Quando registas um domínio, os teus dados pessoais (nome, endereço, e-mail, telefone) ficam disponíveis publicamente na base de dados WHOIS. A Proteção de Privacidade WHOIS substitui esses dados pelos da Webminds, mantendo a tua informação privada. Na Webminds, esta proteção é completamente gratuita e está incluída em todos os domínios.
Sim. Podes transferir o teu domínio de qualquer registador para a Webminds de forma simples. Basta obter o código EPP (Auth Code) junto do teu registador atual e iniciar o processo de transferência no nosso painel. A transferência é rápida e sem tempo de inatividade.
O SSL encripta os dados transmitidos entre o teu site e os visitantes, protegendo informações sensíveis como passwords e dados de pagamento. Além disso, o Google favorece sites com HTTPS nos resultados de pesquisa — o que significa melhor posicionamento SEO. Os navegadores modernos também marcam sites sem SSL como "Não Seguro", o que afasta visitantes.
Um subdomínio é uma extensão do teu domínio principal. Por exemplo, se o teu domínio é empresa.pt, podes criar loja.empresa.pt, blog.empresa.pt ou app.empresa.pt. Na Webminds, podes criar subdomínios ilimitados sem qualquer custo adicional — ideal para separar diferentes secções ou serviços do teu projeto online.
As extensões de domínio (TLDs) indicam o propósito ou localização de um site. O .pt é a extensão nacional de Portugal, ideal para negócios que operam principalmente no mercado português. O .com é a extensão mais reconhecida globalmente, perfeita para negócios internacionais. Existem centenas de outras extensões temáticas como .shop, .tech, .io, .online, entre muitas outras disponíveis na Webminds.
